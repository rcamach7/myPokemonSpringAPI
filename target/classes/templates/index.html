Hello Trainer, Welcome To My Pokemon Catcher Application

  • All Pokemons will show you all your captured pokemons
  • localhost:8080/check/{INPUT_NAME} will let you know if you have captured this pokemon yet
  • localhost:8080/delete/{INPUT_NAME} will release a specific pokemon you have captured.
  • localhost:8080/add/{name}+{type} will let you add a new pokemon given a name and type
